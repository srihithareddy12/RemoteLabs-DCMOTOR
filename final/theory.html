Procedure


Physics of the Experiment

The main physics behind the experiment can be summarized as:


  • We provide input voltage values via our dashboard. Thingspeak serves as the interface for this.
  • We obtain that voltage from Thingspeak and use it to power the motor.
  • We use the rpm sensor to get the motor driver's rotations per minute value and send it to the thingspeak field.
  • We also plot the rpm vs voltage graph to learn about and draw conclusions about the voltage vs rpm characteristic.



Observations

The main physics behind the experiment can be summarized as:


  • speed is directly proportional to the input voltage.
  • The higher the input voltage, the faster the output speed.
  • The lower the input voltage, the slower the output speed.
  • We can control the speed independently of torque by manipulating the supply voltage using a DC motor control unit
